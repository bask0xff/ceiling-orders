# Ceiling Orders 📊🔨

Система автоматизации и управления заказами для бизнеса по производству и установке потолков (натяжных, подвесных, реечных). Проект предназначен для оптимизации процессов обработки заявок, расчета стоимости, контроля этапов выполнения работ и ведения клиентской базы.

## 🚀 Основной функционал

- **Управление заказами:** Создание, редактирование, удаление и отслеживание статусов заказов (в обработке, замер, производство, монтаж, выполнен).
- **Калькулятор стоимости:** Расчет стоимости материалов (полотна, профили, комплектующие) и монтажных работ на основе площади и периметра помещений.
- **База клиентов (CRM):** Хранение контактных данных клиентов, истории их заказов и адресов объектов.
- **Управление сотрудниками:** Назначение замерщиков и монтажных бригад на конкретные объекты с привязкой к календарю.
- **Аналитика и отчеты:** Отслеживание прибыли, популярных материалов и эффективности работы бригад (в разработке / реализовано).

## 🛠 Стек технологий

- **Backend:** Node.js (Express / NestJS) / Python (Django / FastAPI) 
- **Database:** PostgreSQL / MySQL 
- **Frontend:** Vue.js / HTML5 & CSS3
- **Архитектура:** REST API / Docker
